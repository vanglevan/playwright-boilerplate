# 01 — Senior Automation Tester Skills Matrix 2026

> Complete map of skills + proficiency levels expected for senior automation tester roles in 2026.
> Use as self-assessment checklist + learning gap identification.

---

## 📊 Proficiency levels

| Level          | Description                                         |
| -------------- | --------------------------------------------------- |
| **🌱 Aware**   | Know concept exists, can discuss at high level      |
| **🌿 Working** | Can apply with reference/docs, solve known problems |
| **🌳 Fluent**  | Solve novel problems, make architectural choices    |
| **🎓 Expert**  | Teach others, extend tools, contribute to community |

**Senior 2026 = Fluent in 80% of core + Expert in 1-2 specialization areas.**

## 🏁 Senior benchmark — 1 sentence summary

> "A senior automation tester in 2026 is someone who can **design the test strategy** for a team they haven't met, **build an AI agent** to scale the strategy, **mentor junior testers** to execute it, and **measure + communicate ROI** to executives — all while writing better code than they did yesterday."

That's the bar. This curriculum takes you there.
