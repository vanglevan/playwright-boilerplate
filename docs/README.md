## 📘 Curriculum 1: [Playwright Learning Journey (30 ngày)](./playwright-learning-journey/README.md)

**Dành cho:** Manual tester chưa có kinh nghiệm automation.

**Output:** Portfolio GitHub repo với 20+ tests, CI/CD, AI workflow.

### Cấu trúc

| Week | Focus                                                 | Ngày      |
| ---- | ----------------------------------------------------- | --------- |
| 1    | Foundation (JS/TS, Playwright setup, locators, POM)   | Day 1-7   |
| 2    | Core Skills (fixtures, auth, API testing, parallel)   | Day 8-14  |
| 3    | Advanced (visual, a11y, performance, CI/CD, Docker)   | Day 15-21 |
| 4    | AI + Portfolio (Claude Code, MCP, agentic, interview) | Day 22-30 |

### Bạn sẽ học

- Playwright TypeScript end-to-end
- POM, Fixtures, Test Data Factories
- CI/CD với GitHub Actions
- AI-assisted testing (Claude Code, Cursor, MCP)
- Portfolio polish + interview prep

**→ Bắt đầu tại [playwright-learning-journey/README.md](./playwright-learning-journey/README.md)**

## 🌟 Learning principles (áp dụng cho cả 2 curriculum)

1. **70% thực hành, 20% đọc code, 10% lý thuyết** — không watch tutorial quá 30 phút/ngày
2. **Commit daily** lên GitHub — timeline chính là portfolio
3. **Teach to learn** — blog post hoặc share để cố định kiến thức
4. **Measure progress** — tuần/tháng tự review
5. **AI là sensei, không phải ghostwriter** — hỏi "why", không chỉ "how"
6. **Community > cave** — join Discord, ask questions, help others
