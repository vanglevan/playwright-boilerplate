## 20. "When stuck" cheat

Google template searches:

- `playwright "your error message" site:github.com/microsoft/playwright/issues`
- `playwright <topic> 2026` (filter recent)
- `<problem> reddit`

AI prompt template:

```
I'm getting error: [paste]
Context: [Playwright version, what I'm trying to do]
What I tried: [list]
What's the likely root cause + fix?
```
